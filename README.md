# Source

This is the public repository of the source code for the simfuture.quest game (Python) and the underlying model (Vensim). Note, you can clone this repo, but it will not give you a functioning clone. Instead it will give you all the sources one by one so you inspect and study what we did. And use it as a starting point for anthing you may want to do with it further. The license allows you to do this. Happy coding.

To play the game, go to https://simfuture.quest/ If you never played the game, your first click should be on the "About the game" button. From there, follow the link to the user manual.

Enjoy

PS: This is a hobby project, I check this repo for issues very sporadically - while you are waiting for a response, go out into the analog world, enjoy the sun and other people (and your dog).
