# Vensim source code

In this directory you find the files of the model to run them directly in Vensim. For development we used Vensim PRO 10.1.3 You can purchase it [here](https://vensim.com/purchase/). Discounts are available for academic and non-profit use. You can also use the model with the free [Vensim Reader](https://vensim.com/vensim-model-reader/). The limitation is that you cannot save any changes you may make in the model.

You also need to download the EXCEL files, they contain initialization data and policy data. We do believe that you need EXCEL installed on your machine to run the model.

The "*mov251229 ge4a 10reg.mdl*" file **is** the model. It is a text file that can be opened and inspected by any program that reads flat textfiles, like Notepad or Notepad++ on Windows.

Both the *.mdl and the two *.xlsx files **must** reside in the same folder on your local machine. 

## mov251229_ge4a_10reg.txt
This file is the Vensim model file where the variables are sorted in the order of execution. This is useful if you want to use the model as input to a data analysis script you code in Julia, Python, R or some other language.
